# 大括号语言

大括号或大括号编程语言具有使用'{'和'}'字符定义语句区块的语法。这个语法源于BCPL(1966),随着C语言变得广为流行。许多括号语言源自于C或受到C语言的强烈影响。大括号语言的例子包括这些:
